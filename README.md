# tinyfetch

Python and system information command-line fetch tool. Like this!

```
$ tinyfetch

beucismis@thinkpad
------------------
Python Version: 3.11.2
PIP Version: 24.0
PIP Packages: 8
Implementation: CPython
Compiler: GCC 12.2.0

Kernel: Linux-6.1.0-18-amd64
OS: Devuan GNU/Linux 5 (daedalus) x86_64
```

## Installation

```console
pip install tinyfetch
```
